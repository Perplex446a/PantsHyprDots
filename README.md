# PantsHyprDots
These are my Hyprland dotfiles

They are VERY opinionated and simple

If you don't know how to get this on your system, do these simple steps;

1. Make sure you have the git package installed

2. cd into the PantsHyprDots folder: cd ~/PantsHyprDots

3. make sure you have these installed; kitty, waybar, hyprlock, hyprland, hyprpolkitagent, hyprpaper, fireshot, firefox, chromium, fuzzel, dolphin, xdg-desktop-portal-hyprland

4. run this command;
git clone https://github.com/Perplex446a/PantsHyprDots/

5. run this command
chmod +x install.sh

6. run the bash script
./install.sh
